Pebcessing Reference

Processing-like framework for Pebble Smartwatch. Let's sketch a watchface with code!

height

Examples

void draw() {
  background(0);
  noStroke();
  fill(255);
  rect(40, 0, 20, height);
  rect(60, 0, 20, height / 2);
}

Description

System variable that stores the height of the display window. The value of height is ordinarily 168.
